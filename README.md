# Car Price Prediction Model 

A machine learning project designed to predict car prices based on specific features using Regression techniques.

## Project Structure
* **main.py**: The entry point of the application.
* **model.py**: Contains the machine learning model architecture.
* **preprocessor.py**: Handles data cleaning and feature engineering.
* **data_loader.py**: Script for loading and preparing the dataset.
* **car_price_prediction_.csv**: The dataset used for training and testing.
